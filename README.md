## 游戏简介

**PickRubbish** 是一个以 `摸金` `载具` 为核心玩法的游戏，目前没有任何内容。

## 技术栈

- **引擎**：Unreal Engine 5
- **编程语言**：C++ & C# & Blueprint
- **工具**：GitHub 、Blender
